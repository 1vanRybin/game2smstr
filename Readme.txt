Найти выход из лабиринта

Основная идея игры: игра за персонажа с оружием и некоторым набором умений, которые можно использовать в течение
своего хода. Против персонажа выступают NPC.
Создание препятствий для игрока и игроком непосредственно в процессе игры.
Алгоритмы: поиск кратчайшего пути до игрока в лабиринте для npc.

//////////////////////////////////////////////////////////////////////////////////////////
Приветствую тебя, играющий!
Твоя задача - дойти до выхода и не быть убитым монстрами. 
Ты можешь один раз за игру использовать кирку (pick) и разрушить один блок стены(ЛКМ).
Собирай умения и используй их против монстров. 
В правом верхнем углу есть счётчик. 
Это количество патронов, которые ты можешь использовать, чтобы убить монстров(на ПКМ). 
По центру сверху - твои ХП на уровень, следи за ними и постарайся не погибнуть.) 
Успехов!
